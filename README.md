# Creator Platform
「**クリエイター向けTwitter連携型マイクロファンディングプラットフォーム**」

## サービス概要

このサービスは、アダルトコンテンツを扱うクリエイターが、**自身のブランドと独自ドメインで、安全かつ安定的にファンディングサイトを運営できる**ように設計された専門的なプラットフォームです。Twitterを主要な集客チャネルとするクリエイターのニーズに特化し、既存の汎用プラットフォームでは解決できない課題に対応します。

## 主要な特徴と差別化ポイント

### 1. クリエイターの独立性とブランド確立
- **独自ドメインでの運営**: クリエイターは `creatorname.com` のような自身の独自ドメインでファンディングサイトを運営できます。これにより、プラットフォームのブランドに依存せず、クリエイター自身のブランドを前面に出した「公式サイト」としてファンにアピールし、長期的なブランド価値を確立できます。
- **プラットフォーム依存リスクの軽減**: 既存のFantiaやMyFansのようなプラットフォームの規約変更、手数料率の変更、サービス停止といったリスクからクリエイターを保護し、活動の安定性を確保します。

### 2. アダルトコンテンツに特化した機能
- **アダルト対応決済**: アダルトコンテンツの取り扱いが可能な決済プロバイダーと連携し、クリエイターが複数の選択肢から最適な決済手段を選べるようにします。これにより、収益機会の損失を防ぎ、安定した収益化を支援します。
- **厳格な年齢確認**: 未成年者保護のための厳格な年齢確認機能を標準搭載し、クリエイターが法規制を遵守し、安心してコンテンツを提供できる環境を構築します。

### 3. Twitter連携の最適化
- **プロモーション支援**: Twitterでのプロモーションを効率化するため、コンテンツプレビュー（サムネイル画像や短い動画クリップ）の生成機能をMVP段階から提供します。これにより、Twitterのポリシーを遵守しつつ、自身の独自ドメインサイトへファンをスムーズかつ安全に誘導できます。

### 4. テーマ機能で“自分の好きなサイト”を実現（新規）
- **テンプレート選択**:
    - **ライトプラン**：厳選**2**テンプレート
    - **プロプラン**：拡張**4**テンプレート
- **柔軟なカスタマイズ**: カラー／タイポグラフィ／レイアウトオプションをGUIで調整可能。ブランドトーンに合わせたビジュアルをノーコードで作成。
- **ブランド一貫性**: ロゴ・OG画像・SNSカード設定を一括管理し、Twitter投稿からサイト訪問まで一貫した体験を提供。

## 収益モデル

MVP段階では、以下の2つの収益源を組み合わせます。

1. **クリエイターからの月額プラットフォーム利用料**: プラットフォームの安定運営を支える基盤収益。
2. **ファンがクリエイターに支払うサブスクリプション額に対する決済手数料の一部**: クリエイターの成功と共に成長する収益。

- **ライトプラン**：月額980円 + 手数料10%
- **プロプラン**：月額2,980円 + 手数料8%

このサービスは、アダルトコンテンツクリエイターが直面する「プラットフォーム依存」「決済問題」「法規制対応」「Twitterからの誘導効率化」といった複合的な課題を解決し、クリエイターがより自律的に、安全に、そして効率的に活動し、自身のブランド価値を高めながら収益を最大化できる「自分だけの場所」を提供するSaaSです。

## ローカル環境でのNext.js画面確認手順

1. ルート `Creator` ディレクトリで依存関係をインストール  
   ```bash
   npm install
   ```
2. 同じ場所でNext.js開発サーバーを起動  
   ```bash
   npm run dev:web
   ```
3. ブラウザで `http://localhost:3000/creators/login` または `http://localhost:3000/creators/signup` にアクセスすると、`apps/web/app/creators/*` のページ内容をリアルタイムで確認できます。
