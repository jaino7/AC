# テーマプレビュー画像

このディレクトリには6つのテーマのプレビュー画像を配置します。

## 必要な画像ファイル

1. `creator-pro.jpg` - Creator Pro テーマ
2. `neon-pro.jpg` - Neon Pro テーマ
3. `studio-pro.jpg` - Studio Pro テーマ
4. `velvet-pro.jpg` - Velvet Pro テーマ
5. `pure-lite.jpg` - Pure Lite テーマ
6. `zine-lite.jpg` - Zine Lite テーマ

## 推奨仕様

- **サイズ**: 800x600px 以上
- **フォーマット**: JPG または WebP
- **アスペクト比**: 4:3
- **内容**: 各テーマの実際のページプレビュー（デスクトップ表示）

## 一時的なプレースホルダー

実際の画像を用意するまで、以下の方法でプレースホルダーを使用できます：

### 1. Unsplash のプレースホルダー
- URL: `https://source.unsplash.com/800x600/?theme,website`

### 2. プレースホルダーサービス
- https://placehold.co/800x600/png

### 3. 実際のテーマページのスクリーンショット
各テーマページにアクセスしてスクリーンショットを撮影：
- `/creator-pro`
- `/neon-pro`
- `/studio-pro`
- `/velvet-pro`
- `/pure-lite`
- `/zine-lite`

## 注意事項

画像が存在しない場合、Next.js の Image コンポーネントがエラーを表示します。
必ず6つすべての画像ファイルを配置してください。
